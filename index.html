Projects

— Project 01
PixelPhantom
v2.0.0  ·  90/90 tests passing  · 

A desktop application for detecting and removing duplicate image, video, and audio files using a novel pixel-direct comparison approach — no hashing at any stage. Two files are only flagged as duplicates if their actual decoded pixel data is identical, making it immune to re-encoding artifacts that fool MD5/SHA-based tools.

Built with a three-phase scan pipeline: a size gate eliminates ~90% of files instantly, a dimension gate handles another layer, and only remaining files undergo full pixel comparison via Pillow's tobytes(). All scanning runs in background daemon threads — the UI never blocks. Files go to a recycle bin by default, never permanently deleted without confirmation.
